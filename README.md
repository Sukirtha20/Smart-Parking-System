# 🚗 Smart Parking System 

A web-based Smart Parking Management System developed using **Python, Flask, HTML, CSS, JavaScript, and SQLite**. The system automates vehicle entry and exit, parking slot management, fee calculation, loyalty-based discounts, and WhatsApp notifications.

---

## 📌 Project Overview

The Smart Parking System is designed to simplify parking management by replacing manual processes with an automated web application. It stores customer and vehicle details digitally, calculates parking fees based on parking duration, and rewards regular customers through a loyalty pricing feature.

---

## ✨ Features

- 🚘 Vehicle Entry & Exit Management
- 🅿️ Parking Slot Allocation
- 🎟️ Automatic Token Generation
- ⏱️ Time-Based Parking Fee Calculation
- 💰 Loyalty Pricing for Regular Customers
- 🗄️ SQLite Database Storage
- 📱 WhatsApp Notification

## ⚙️ System Workflow

1. User enters vehicle details.
2. System checks parking slot availability.
3. A unique parking token is generated.
4. Entry details are stored in the SQLite database.
5. During exit, parking duration is calculated.
6. Parking fee is generated automatically.
7. Loyalty discount is applied for regular customers.
8. WhatsApp notification is sent to the customer.

## 🚀 Installation

1. Clone the repository

```bash
git clone https://github.com/sukirtha20/Smart-Parking-System.git
```

2. Navigate to the project

```bash
cd SmartParkingSystem
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Run the application

```bash
python app.py
```

5. Open your browser

```
http://127.0.0.1:5000
```
