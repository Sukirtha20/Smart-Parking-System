## вҡҷпёҸ System Workflow

1. User enters vehicle details.
2. System checks parking slot availability.
3. A unique parking token is generated.
4. Entry details are stored in the SQLite database.
5. During exit, parking duration is calculated.
6. Parking fee is generated automatically.
7. Loyalty discount is applied for regular customers.
8. WhatsApp notification is sent to the customer.
